# Fashion MNIST Image Classification

## Overview

This project builds a Convolutional Neural Network (CNN) to classify clothing images from the Fashion MNIST dataset. The model predicts the category of clothing such as T-shirt, shoe, bag, or coat.

## Technologies

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Streamlit

## Dataset

The Fashion MNIST dataset contains **70,000 grayscale images (28x28 pixels)** across **10 clothing categories**.

## Model

A CNN model is used to automatically extract image features and classify the clothing items with high accuracy.
